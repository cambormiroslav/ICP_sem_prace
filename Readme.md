# ICP: OPENGL

## Keyboard usings

* movement: `W`, `A`, `S`, `D`
* play music: `space`
* turn on or off: `spotlight`
* fullscreen swither: `F11`
* turn on or off imgui: `I`
* turn on or off imgui: `V`
* end the app: `ESC`
* spotlight intensity: `in imgui`

## Spotlight location

* in back of the glass cube
